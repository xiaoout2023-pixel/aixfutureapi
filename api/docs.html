数据模型

每个模型对象包含以下字段：

model_id: 唯一标识（string，如 openai-gpt-4o, deepseek-deepseek-chat）
model_name: 模型名称（string，如 GPT-4o, DeepSeek V3）
provider: 厂商（string，如 openai / google / deepseek / aliyun / anthropic）
release_date: 发布时间（YYYY-MM-DD 或 null）
status: 状态（active / beta / deprecated）

capabilities: 能力对象
  text: 文本生成（bool）
  vision: 图像理解（bool）
  audio: 音频能力（bool）
  code: 代码生成（bool）
  reasoning: 推理能力（bool）
  tool_use: 工具调用（bool）
  function_calling: 函数调用（bool）
  image_generation: 图像生成（bool）
  video_understanding: 视频理解（bool）
  video_generation: 视频生成（bool）
  json_mode: JSON模式（bool）
  structured_output: 结构化输出（bool）
  code_execution: 代码执行（bool）
  fine_tuning: 微调支持（bool）
  embedding: 嵌入模型（bool）
  context_length: 最大上下文长度（int，如 128000）
  max_output_tokens: 最大输出token数（int，如 4096）
  reasoning_level: 推理等级（low/medium/high）

pricing: 定价对象
  input_per_1m_tokens: 输入价格（每100万token，USD）
  output_per_1m_tokens: 输出价格（每100万token，USD）
  cached_input_price: 缓存输入价格（每100万token，USD 或 null）
  batch_input_price: 批量输入价格（USD 或 null）
  batch_output_price: 批量输出价格（USD 或 null）
  price_per_image: 每张图片价格（USD 或 null）
  price_per_request: 每次请求价格（USD 或 null）
  reasoning_price_per_1m: 推理价格（每100万token，USD 或 null）
  currency: 货币（默认USD）
  free_tier: 是否免费（bool）

scores: 评分对象
  reasoning_score: 推理评分（0-100）
  coding_score: 编码评分（0-100）
  speed_score: 速度评分（0-100）
  cost_efficiency_score: 性价比评分（0-100）
  overall_score: 综合评分（0-100）
  latency_level: 延迟等级（low/medium/high）
  throughput_level: 吞吐等级（low/medium/high）

tags: 标签数组（如 ["vision", "coding", "tool_use", "reasoning", "cheap", "premium", "long_context"]）

source: 来源对象
  model_page: 模型页面URL
  api_docs: API文档URL
  pricing_page: 定价页面URL
  last_updated: 最后更新日期
  source_type: 数据来源类型（official / marketplace / open_source）
  region_restriction: 是否有地区限制（bool）
  enterprise_only: 是否仅企业可用（bool）
  openai_compatible: 是否兼容OpenAI API（bool）
  sdk_support: 是否提供SDK（bool）

GET /api/models

模型列表。

参数

参数类型说明
providerstring按厂商筛选，如 openai
statusstring按状态筛选，如 active
tagsstring按标签筛选，逗号分隔，如 vision,coding
typestring按类型筛选，llm/multimodal/vision/audio/code
accessstring按开源类型筛选，open/closed
min_contextint最小上下文长度
max_input_pricefloat最大输入价格（每百万tokens）
max_output_pricefloat最大输出价格（每百万tokens）
has_visionbool是否支持视觉，true/false
has_tool_callingbool是否支持工具调用，true/false
sort_bystring排序字段：overall_score/cost_efficiency_score/input_price/output_price/context_length
sort_orderstring排序方向：asc/desc，默认 desc
pageint页码，默认 1
page_sizeint每页数量，默认 20，最大 100

响应

{
  "code": 200,
  "message": "success",
  "data": [模型对象数组],
  "total": 总数量,
  "page": 当前页码,
  "page_size": 每页数量,
  "total_pages": 总页数
}

示例

/api/models?provider=openai
/api/models?type=multimodal
/api/models?access=closed
/api/models?page=1&page_size=10

GET /api/calculator/scenarios

场景成本模拟器。支持多步骤 AI 调用流程的成本计算（TCO）。

场景管理

方法路径说明
POST/api/calculator/scenarios新建场景，请求体 {"name": "场景名"}
GET/api/calculator/scenarios场景列表
GET/api/calculator/scenarios/{scenario_id}获取场景详情（含步骤和成本汇总）
PUT/api/calculator/scenarios/{scenario_id}更新场景名称
DELETE/api/calculator/scenarios/{scenario_id}删除场景
POST/api/calculator/scenarios/{scenario_id}/duplicate复制场景

步骤管理

方法路径说明
POST/api/calculator/scenarios/{scenario_id}/steps添加步骤
GET/api/calculator/scenarios/{scenario_id}/steps获取步骤列表
PUT/api/calculator/steps/{step_id}更新步骤
DELETE/api/calculator/steps/{step_id}删除步骤
POST/api/calculator/scenarios/{scenario_id}/reorder调整步骤顺序

步骤字段

字段类型说明
task_typestring任务类型，如"意图识别"、"内容生成"
model_idstring模型 ID
input_tokensint输入 token 数量
output_tokensint输出 token 数量
daily_callsint调用量/天
cache_hit_ratefloat缓存命中率，0-100

成本计算公式

指标公式
单次成本(输入tokens × 输入单价 + 输出tokens × 输出单价) / 1,000,000
日成本单次成本 × 调用量 × (1 - 缓存命中率)
月成本日成本 × 30
年成本月成本 × 12

场景对比

方法路径说明
POST/api/calculator/compare对比多个场景，请求体 ["场景ID1", "场景ID2"]

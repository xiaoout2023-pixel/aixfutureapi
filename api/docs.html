AI Model Pricing API

AI模型价格采集与对比系统 v2.0.0 — https://www.aixfutrueapi.top

基础接口

数据模型 GET /api/models GET /api/models/:id GET /api/models/:id/pricing GET /api/models/:id/evaluations GET /api/providers GET /api/search GET /api/compare GET /api/status GET /api/exchange-rate

排行榜

GET /api/leaderboard GET /api/leaderboard/categories GET /api/leaderboard/superclue/:category

成本计算

POST /api/cost/calculate POST /api/cost/compare

数据模型

基于 models / pricing / evaluations 三表联合查询，每个模型对象包含以下字段：

model_id: 唯一标识（string，如 openai/gpt-4o-2024-08-06, anthropic/claude-opus-4-7）
model_name: 模型名称（string，如 GPT-4o, claude-opus-4-7）
provider: 厂商（string，如 openai / google / deepseek / anthropic）
provider_type: 厂商类型（open_source / closed）
release_date: 发布时间（YYYY-MM-DD 或 null）
status: 状态（active / beta / deprecated / coming_soon）
aliases: 别名列表（string[]）
capabilities: 能力对象
  text: 文本生成（bool）
  code: 代码生成（bool）
  reasoning: 推理能力（bool）
  vision: 图像理解（bool）
  image_gen: 图像生成（bool）
  audio: 音频理解（bool）
  audio_gen: 音频生成（bool）
  video: 视频理解（bool）
  tool_use: 工具调用（bool）
  structured_output: 结构化输出（bool）
  streaming: 流式输出（bool）
  batch: 批处理（bool）
  fine_tuning: 微调支持（bool）
  embedding: 向量嵌入（bool）
context_length: 最大上下文长度（int，如 128000）
max_output_tokens: 最大输出token数（int，如 16384）
regions: 支持区域（string[]，如 ["us", "eu", "ap"]，空=全球）
private_deployment: 是否支持私有部署（bool）
openai_compatible: 是否兼容OpenAI API（bool）
urls: 相关链接对象
  official: 官方介绍页
  docs: API文档页
  pricing: 定价页
tags: 标签数组（如 ["flagship", "vision", "coding"]）
last_updated: 最后更新时间（ISO 8601）

pricing: 最新官方全球定价对象
  input_per_1m_tokens: 输入价格（每100万token，USD）
  output_per_1m_tokens: 输出价格（每100万token，USD）
  cache_read_per_1m: 缓存读取价格（每100万token，USD 或 null）
  cache_write_per_1m: 缓存写入价格（每100万token，USD 或 null）
  currency: 货币（默认USD）
  free_tier_tokens: 免费额度（token数/月 或 null）
  reasoning_tokens_charged: 是否对推理token计费（bool）

evaluation: 最新评测数据对象（合并 AA + LMArena）
  aa_intelligence_index: AA综合智能评分（0-100）
  aa_coding_index: AA代码能力评分（0-100）
  aa_math_index: AA数学能力评分（0-100）
  tokens_per_second: 输出速度（tokens/秒）
  ttft_ms: 首Token延迟（毫秒）
  lmarena_elo: LMArena综合对战Elo（约500-2000）
  lmarena_coding: LMArena代码对战Elo
  lmarena_math: LMArena数学对战Elo
  lmarena_hard: LMArena难题对战Elo

GET /api/models

模型列表，支持多维度筛选和排序。数据来自 models + pricing + evaluations 三表联合查询。

参数

参数类型说明
providerstring按厂商筛选，如 openai, anthropic, google
provider_typestring按厂商类型筛选：open_source / closed
capabilitystring按能力筛选：text/code/reasoning/vision/image_gen/audio/audio_gen/video/tool_use/structured_output/streaming/batch/fine_tuning/embedding
statusstring按状态筛选：active/beta/deprecated/coming_soon
qstring搜索关键词（模型ID、名称、厂商）
sort_bystring排序字段：aa_intelligence_index / lmarena_elo / input_price / output_price / context_length / tokens_per_second
sort_orderstring排序方向：asc / desc，默认 desc
min_input_pricefloat最小输入价格（每百万tokens）
max_input_pricefloat最大输入价格（每百万tokens）
pageint页码，默认 1
page_sizeint每页数量，默认 20，最大 100

响应

{
  "code": 200,
  "message": "success",
  "data": [模型对象数组],
  "total": 总数量,
  "page": 当前页码,
  "page_size": 每页数量,
  "total_pages": 总页数
}

示例

/api/models?provider=openai
/api/models?capability=vision
/api/models?provider_type=closed
/api/models?sort_by=lmarena_elo
/api/models?min_input_price=0&max_input_price=5
/api/models?q=claude&page=1&page_size=10

GET /api/compare

参数

参数类型必填说明
modelsstring是逗号分隔的模型ID，最多10个，如 openai/gpt-4o-2024-08-06,anthropic/claude-opus-4-7

响应

{
  "code": 200,
  "message": "success",
  "data": [模型对象数组],
  "meta": {
    "cheapest_input": "openai/gpt-4o-2024-08-06",
    "cheapest_output": "openai/gpt-4o-2024-08-06",
    "best_intelligence": "anthropic/claude-opus-4-7",
    "longest_context": "anthropic/claude-opus-4-7"
  }
}

POST /api/cost/calculate

单个模型成本计算。

请求体

{
  "model_id": "openai/gpt-4o-2024-08-06",
  "input_tokens": 100000,
  "output_tokens": 50000,
  "quantity": 10,
  "currency": "CNY"
}

参数

参数类型必填说明
model_idstring是模型ID
input_tokensint否输入token数量，默认1000
output_tokensint否输出token数量，默认1000
quantityint否调用次数，默认1
currencystring否货币 USD/CNY/EUR/JPY/GBP，默认USD

响应

{
  "code": 200,
  "message": "success",
  "data": {
    "model_id": "openai/gpt-4o-2024-08-06",
    "model_name": "OpenAI: GPT-4o (2024-08-06)",
    "provider": "openai",
    "input_tokens": 100000,
    "output_tokens": 50000,
    "quantity": 10,
    "pricing": { ... },
    "cost_breakdown": {
      "input_cost": 1.8125,
      "output_cost": 3.625,
      "total_cost": 5.4375,
      "currency": "CNY",
      "currency_symbol": "¥",
      "exchange_rate": 7.25
    }
  }
}

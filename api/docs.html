AI Model Pricing API

AI模型价格采集与对比系统 v0.3.0 — https://www.aixfutrueapi.top

基础接口

数据模型 GET /api/models GET /api/models/:id GET /api/compare GET /api/providers GET /api/search GET /api/search/suggest GET /api/status GET /api/exchange-rate GET /api/model-types

成本计算

POST /api/cost/calculate POST /api/cost/compare

场景模拟器

GET /api/calculator/scenarios GET /api/calculator/templates

排行榜

GET /api/leaderboard GET /api/leaderboard/periods GET /api/leaderboard/summary

数据模型

每个模型对象包含以下字段：

model_id: 唯一标识（string，如 gpt-4o）
model_name: 模型名称（string，如 GPT-4o）
provider: 厂商（string，如 openai / aliyun / anthropic）
release_date: 发布时间（YYYY-MM-DD）
status: 状态（active / beta / deprecated）

capabilities: 能力
  text_generation: 文本生成（true/false）
  code_generation: 代码生成（true/false）
  vision: 图像理解（true/false）
  audio: 音频能力（true/false）
  multimodal: 多模态（true/false）
  tool_calling: 工具调用（true/false）
  context_length: 最大上下文长度（数字）
  reasoning_level: 推理等级（low/medium/high）

pricing: 定价
  input_price_per_1m_tokens: 输入价格（每100万token，USD）
  output_price_per_1m_tokens: 输出价格（每100万token，USD）
  currency: 货币（默认USD）

scores: 评分
  overall_score: 综合评分（0-100）
  cost_efficiency_score: 性价比评分（0-100）

tags: 标签数组（如 ["多模态", "coding", "vision"]）
source: 来源信息

GET /api/models

模型列表。

参数

参数类型说明
providerstring按厂商筛选，如 openai
statusstring按状态筛选，如 active
tagsstring按标签筛选，逗号分隔，如 vision,coding
typestring按类型筛选，llm/multimodal/vision/audio/code
accessstring按开源类型筛选，open/closed
min_contextint最小上下文长度
max_input_pricefloat最大输入价格（每百万tokens）
max_output_pricefloat最大输出价格（每百万tokens）
has_visionbool是否支持视觉，true/false
has_tool_callingbool是否支持工具调用，true/false
sort_bystring排序字段：overall_score/cost_efficiency_score/input_price/output_price/context_length
sort_orderstring排序方向：asc/desc，默认 desc
pageint页码，默认 1
page_sizeint每页数量，默认 20，最大 100

响应

{
  "code": 200,
  "message": "success",
  "data": [模型对象数组],
  "total": 总数量,
  "page": 当前页码,
  "page_size": 每页数量,
  "total_pages": 总页数
}

示例

/api/models?provider=openai
/api/models?type=multimodal
/api/models?access=closed
/api/models?page=1&page_size=10

text_generationbool文本生成，true/false
code_generationbool代码生成，true/false
visionbool视觉，true/false
audiobool音频，true/false
multimodalbool多模态，true/false
tool_callingbool工具调用，true/false
reasoning_levelstring推理等级，low/medium/high
min_input_pricenumber输入价格下限
max_input_pricenumber输入价格上限
min_output_pricenumber输出价格下限

sort_bystring排序字段：overall_score/cost_efficiency_score/input_price/output_price/context_length（默认 cost_efficiency_score）

GET /api/calculator/scenarios

场景成本模拟器。支持多步骤 AI 调用流程的成本计算（TCO）。

场景管理

方法路径说明
POST/api/calculator/scenarios新建场景，请求体 {"name": "场景名"}
GET/api/calculator/scenarios场景列表
GET/api/calculator/scenarios/{scenario_id}获取场景详情（含步骤和成本汇总）
PUT/api/calculator/scenarios/{scenario_id}更新场景名称
DELETE/api/calculator/scenarios/{scenario_id}删除场景
POST/api/calculator/scenarios/{scenario_id}/duplicate复制场景

步骤管理

方法路径说明
POST/api/calculator/scenarios/{scenario_id}/steps添加步骤
GET/api/calculator/scenarios/{scenario_id}/steps获取步骤列表
PUT/api/calculator/steps/{step_id}更新步骤
DELETE/api/calculator/steps/{step_id}删除步骤
POST/api/calculator/scenarios/{scenario_id}/reorder调整步骤顺序

步骤字段

字段类型说明
task_typestring任务类型，如"意图识别"、"内容生成"
model_idstring模型 ID
input_tokensint输入 token 数量
output_tokensint输出 token 数量
daily_callsint调用量/天
cache_hit_ratefloat缓存命中率，0-100

成本计算公式

指标公式
单次成本(输入tokens × 输入单价 + 输出tokens × 输出单价) / 1,000,000
日成本单次成本 × 调用量 × (1 - 缓存命中率)
月成本日成本 × 30
年成本月成本 × 12

场景对比

方法路径说明
POST/api/calculator/compare对比多个场景，请求体 ["场景ID1", "场景ID2"]

GET /api/leaderboard

获取排行榜数据。支持通用榜和多模态榜。

参数

参数类型必填说明
board_typestring是榜单类型：general(通用榜) / multimodal(多模态榜)
periodstring否评测周期，如 2026-03，默认最新
providerstring否按厂商筛选
sort_bystring否排序字段：rank/score/generation_time/composite_price，默认 rank
sort_orderstring否排序方向：asc/desc，默认 asc
pageint否页码，默认 1
page_sizeint否每页数量，默认 20，最大 100

        "math_reasoning": 92.1,

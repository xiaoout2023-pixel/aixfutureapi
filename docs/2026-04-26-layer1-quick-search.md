
# 第一层：快速查价 — 产品功能文档

> 状态：待确认 | 日期：2026-04-26

---

## 1. 定位

用户进入成本计算页面，5 秒内回答"什么模型便宜又好用"。
这是最轻量的入口，也是第二层的"流量池"。

---

## 2. 搜索与筛选

### 2.1 搜索

| 功能 | 说明 |
|------|------|
| 模型名称搜索 | 输入 "gpt-4o"、"claude" "deepseek"，模糊匹配，即时出结果 |
| 任务类型搜索 | 输入 "文本分类"、"代码生成"、"翻译"，匹配能力标签 |
| 厂商筛选 | 多选：OpenAI / Anthropic / Google / DeepSeek / ... |
| 价格区间 | 滑块或输入框，按输入价格或输出价格过滤 |
| 排序切换 | 默认按"性价比"排，可切换为"价格从低到高"、"能力分从高到低" |

### 2.2 关键逻辑

- **性价比计算**：综合能力分 ÷ 综合价格 = 性价比指数。综合能力分取该模型各维度分数的加权平均；综合价格取输入/输出均价。
- **搜索联想**：用户输入时，同时匹配模型名、厂商名、能力标签三类关键词，去重展示。

---

## 3. 模型列表卡片

每个模型一张卡片，展示：

| 字段 | 说明 |
|------|------|
| 模型名称 + 厂商 logo | 一眼辨识 |
| 性价比排名标签 | 如"🏆 分类任务性价比第1" |
| 核心价格 | 输入 $X/1M tokens 、输出 $Y/1M tokens |
| 能力分亮点 | 取该模型分数最高的 2-3 项能力展示（如 文本生成 92 · 推理 88） |
| 操作按钮 | "查看详情"、"加入对比"、"加入场景模拟" |

### 3.1 卡片展开

点击卡片展开，显示完整价格表：

| 价格类型 | 说明 |
|---------|------|
| 标准输入价 | 每 1M tokens |
| 标准输出价 | 每 1M tokens |
| 缓存输入价 | prompt caching 命中时（如有） |
| Batch 价 | 批量处理价（如有） |
| 微调价 | 如有 |
| 最后更新时间 | 价格更新时间戳 |

---

## 4. 模型对比

- 勾选 2-4 个模型（复选框）
- 点击"对比"或顶部浮现对比栏
- 并排表格展示：模型名、输入价、输出价、各能力分、性价比指数
- 差异高亮：最优值绿色高亮，最差值红色标记
- 支持清空对比、移除单个

---

## 5. 交互细节

### 5.1 空状态
- 无搜索结果时：展示热门搜索标签、推荐模型

### 5.2 加载态
- 骨架屏卡片占位，不用全屏 loading

### 5.3 错误态
- 数据加载失败：重试按钮 + "使用缓存数据"降级方案

### 5.4 响应式
- 移动端：卡片纵向排列，筛选收起到顶部抽屉
- 桌面端：左侧筛选栏 + 右侧卡片网格

---

## 6. 与第二层的衔接

- 每个模型卡片上的"加入场景模拟"按钮
- 对比栏也有"将对比结果加入场景模拟"
- 跳转至第二层时，预填所选模型信息
